# 开发须知

## 安装必要的软件
 - node(版本大于4)
 - git

## 安装包管理器和依赖库
  在已经安装好Node的前提下，首先先全局安装yarn包管理器
  ```
  npm install -g yarn
  ```

  然后，安装依赖库
  ```
  yarn
  ```

## 命令须知
  ```
  npm run dev  ==> 是开发环境的webpack命令，实时监控打包
  npm run build ==> 是生产环境的webpack命令，只打包一次
  ```

## 技术须知
